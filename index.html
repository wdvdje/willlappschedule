      🔄 Sync Between Devices (GitHub)

      Keeps your data in sync across your computer and iPhone using your existing GitHub account. Takes about 30 seconds to set up — no new accounts needed.

      How to set up (one-time)
      1. Open github.com/settings/personal-access-tokens/new
      2. Give it any name (e.g. TimeScape Sync), set an expiration
      3. Under Account permissions → Gists, choose Read and write
      4. Click Generate token and copy it
      5. Paste the token below and click Save & Sync
      6. On your iPhone: open Settings here, paste the same token, click Save & Sync
