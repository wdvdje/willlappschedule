    ⚡ Auto-import via iOS Shortcut

    Set up a Shortcut that automatically reads timescape-data.json from your iCloud Drive and loads it into the app — no manual picking required.

    1. Open the Shortcuts app and create a new Shortcut.
    2. Add action: Get File → choose the file timescape-data.json from your iCloud Drive (e.g. iCloud Drive / Shortcuts / timescape-data.json). Enable "Show file picker" once to locate it, then disable it so future runs grab it automatically.
    3. Add action: Encode → set encoding to Base64.
    4. Add action: Copy to Clipboard — this copies the Base64 output from the previous step to your clipboard.
    5. Add action: URL → paste the URL below exactly as-is (do not modify it — the data travels via clipboard, not the URL).
    6. Add action: Open URLs.
    7. Run the Shortcut — the app will open, read the clipboard, and import your data automatically. Tap Allow if asked for clipboard permission.

    💡 Tip: Save the exported file as timescape-data.json in an iCloud Drive folder of your choice and point your Shortcut at that path. Run the Export button above first to create the initial file.

    ℹ️ iCloud Drive sync is not available in this browser. For best results, use Safari on macOS or iOS.
