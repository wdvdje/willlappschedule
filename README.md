# willlappschedule
A scheduling app for stupids.

## Running locally

Open `index.html` in a browser, or serve the files with any static HTTP server.

## Sync between devices

The app supports syncing data between devices via **GitHub Gist**. No extra server or account needed — just a GitHub Personal Access Token with the `gist` scope. See Settings → Sync Between Devices (GitHub) in the app for setup instructions.
